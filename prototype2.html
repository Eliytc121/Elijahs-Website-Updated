Description

Prototype 2 included improvements based on testing and user feedback. We changed the top for easier access with the buttons and fitting the actual top on the case and with a cleaner look.

Changes Made

  • Button access was hard to press and access the buttons.
  • Improved top board fit — it was hard to put on and take off, and nearly broke.
  • Cleaner appearance — removed random back bump for better design.
  • Made the ventilation holes bigger.

Pros

  • Lid fits much better
  • Cleaner look
  • Easy button access
  • Better ventilation

Cons / Problems

  • Needs improved appearance

Planned Improvements

  • Improve overall appearance
